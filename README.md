\# Proyecto: Cafetín Central (Fase II)



Este proyecto es un \*\*prototipo funcional\*\* desarrollado para la asignatura \*\*Introducción a la Informática (6301)\*\* de la Escuela de Computación de la Universidad Central de Venezuela (Semestre 2-2025).



\##  Descripción del Proyecto

El sistema simula las principales interacciones digitales de un cafetín universitario, permitiendo la gestión de pedidos, administración de menú y registro de ventas mediante una interfaz web intuitiva y dinámica.



\##  Funcionalidades Implementadas

El prototipo cuenta con tres módulos principales basados en roles de usuario:



1\. \*\*Módulo de Cliente:\*\*

&nbsp;  - Visualización dinámica del menú de alimentos y bebidas.

&nbsp;  - Selección de productos con cálculo automático del total a pagar en tiempo real.

2\. \*\*Módulo de Personal de Caja:\*\*

&nbsp;  - Simulación de punto de venta para el procesamiento de cobros y finalización de transacciones.

3\. \*\*Módulo de Administrador:\*\*

&nbsp;  - Gestión estática del menú (agregar nuevos productos al sistema).

4\. \*\*Sistema de Seguridad:\*\*

&nbsp;  - Control de acceso mediante login con validación de credenciales y redirección por roles.



\##  Tecnologías Utilizadas

\- \*\*HTML5:\*\* Estructura de las interfaces de usuario.

\- \*\*CSS3:\*\* Diseño visual personalizado con identidad corporativa (estilo cafetín).

\- \*\*JavaScript (Vanilla):\*\* Lógica de negocio, manipulación del DOM, manejo de arreglos de objetos y validación de usuarios.

\- \*\*Git \& GitHub:\*\* Control de versiones y gestión del repositorio.

\- `3.0 index.html`: Pantalla de inicio de sesión.

\- `administrador.html`: Interfaz de gestión administrativa.
